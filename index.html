📍 Live Tracker App - Handleiding

🚀 Hoe te starten?

Optie 1: Master (Jij deelt je locatie)

  • Klik "START TRACKING"
  • Je krijgt een 4-cijfer code
  • Klik "COPY CODE" en stuur via WhatsApp/SMS naar je vrienden
  • Je locatie wordt gedeeld in real-time

Optie 2: Follower (Volgen)

  • Ontvang de 4-cijfer code
  • Typ de code in het invoerveld
  • Je ziet de Master direct op de kaart!

🗺️ Kaart-opties

CENTER: ON/OFF - Kaart volgt je locatie automatisch

ZOOM slider (12-18) - Zoom in/uit

GO knop (onderaan) - Verbergt alle controls, alleen kaart zichtbaar

📊 GPX bestanden (routes)

LOAD GPX - Laad een route bestand

SAVE GPX - Menu met 3 opties:

  • Save master track (rood - Master's live tracking)
  • Save master loaded track (cyaan - Master's ingeladen route)
  • Save follower track (blauw - Jouw route, alleen in Follower mode)

👥 Follower-modus specials

Blauwe tracking lijn - Je ziet een blauwe lijn waar jij bent gereden

ZOOM-modus (3 opties):

  • ZOOM: MASTER - Volg Master
  • ZOOM: BOTH - Beide in beeld
  • ZOOM: SELF - Jouw locatie

Track-knop (3 opties):

  • HIDE TRACK - Geen tracking zichtbaar
  • VIEW TRACK - Zie Master's tracking (rode lijn)
  • ALL TRACK - Zie tracking + Master's ingeladen route (cyaan lijn)

👥 Multi-user (meerdere volgers)

Meerdere mensen kunnen dezelfde 4-cijfer code gebruiken en dezelfde Master volgen tegelijk.

Geen limiet in volgers, maar Firebase database kan overbelast raken bij honderden volgers.

Voor normale groepen (5-10 volgers) werkt het prima.

⚙️ Tips

  • Houd je scherm OPEN - de app werkt niet op achtergrond
  • GPS werkt best buiten / in open ruimte
  • Langzaam internet? Tracking update elke 3 seconden
